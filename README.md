# 🚆 SwiftRail

SwiftRail is a modern train ticket booking web application inspired by IRCTC.

## 🌐 Live Demo
https://swift-rail-pnw9.vercel.app

## Features
- Search trains between cities
- Passenger booking flow
- Ticket generation
- QR code ticket verification
- Mobile-friendly UI

## Tech Stack
- Next.js
- React
- Tailwind CSS
- Node.js
- Express
